## Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/Semutireng22/pixelfarm.git
   cd pixelfarm
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Set up your account ID:

   Place your account ID in a file named `data.txt`.

4. Run the script:

   ```bash
   python3 main.py
   ```
